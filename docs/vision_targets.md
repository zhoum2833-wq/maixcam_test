# 视觉系统性能指标

## 帧率分级

| 帧率 | 效果 |
|------|------|
| < 60Hz | 动态追踪滞后、震荡，高速移动靶难以收敛，高分难度极大 |
| **80~100Hz** | **最优甜点**，配合卡尔曼补偿一帧延迟，动态跟踪、静态瞄准都能稳住高精度 |
| > 100Hz | MaixCAM Pro 很难稳定长时间跑满，强行拉高会出现波动、丢帧，收益有限 |

## 稳定性 > 峰值帧率

```
稳定 85Hz 持续运行  >  峰值 100Hz、时常掉到 50Hz
```

帧率抖动 = 控制回路采样周期忽长忽短，PID 极易震荡、打靶偏移。

## 时延指标

链路延迟 = 摄像头曝光 → 图像读取 → 图像处理 → 串口发送 → 单片机接收

同样 100Hz，一帧延迟 12ms 和 22ms，控制效果天差地别。必须依靠卡尔曼 / 前馈补偿这部分滞后。

## 电机控制参考

| 层级 | 频率 | 说明 |
|------|------|------|
| 视觉外环 | 80~100Hz | 视觉帧率，串口跟随发送 |
| 电机内环 | 500~1000Hz | 单片机编码器闭环周期，不是串口接收频率 |

内环持续高频运行，两次视觉指令之间靠电机闭环维持稳定。

## 四大核心目标（优先级排序）

1. **稳定性第一** — 帧率平稳，不剧烈波动
2. **时延指标** — 测出整套链路延迟
3. **帧率摸底** — 找到算法能稳定跑住的最高帧率
4. **识别鲁棒性** — 光线变化、远近、遮挡下不丢目标、不跳坐标
